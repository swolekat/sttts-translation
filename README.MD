# Speech to Text to Speech Translation edition

This is a small app that hosts a webpage that will use chrome's voice recognition and speech synthesis apis to translate what you say into english and say it.

Default language it's translating is lithuanian but you can change it to whatever.

## How to run
* download the exe from the latest release
* run the exe
* in a **CHROME** browser go to http://localhost:57775/
* allow microphone
* speak
